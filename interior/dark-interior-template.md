# Dark Interior Style Guide

This document defines the CSS/Style rules for the "From Storm to Fire" ebook production.

## Color Palette
- **Primary Background:** `#121212` (Charcoal/Black)
- **Primary Text:** `#E0E0E0` (Off-white)
- **Heading 1 (Chapter Titles):** `#FF8C00` (Dark Orange/Fire)
- **Heading 2/3:** `#FFD700` (Gold)
- **Accents/Borders:** `#FFD700` (Gold)

## Typography
- **Body Text:** Serif font (e.g., Georgia, Crimson Text) for readability.
- **Headings:** Sans-serif or Decorative (e.g., Cinzel, Playfair Display).

## Layout Rules
1. **Chapter Openers:** 
   - Full-page background color.
   - Centered title in Gold/Orange.
   - Gold border (1pt) around the page margin.
2. **Body Pages:**
   - Clean charcoal background.
   - Sufficient margins (1 inch) for readability.
   - Page numbers centered at the bottom in light grey.
